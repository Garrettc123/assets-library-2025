# Contributing Guidelines

## Adding Assets

1. Place assets in correct category
2. Use descriptive filenames
3. Optimize file sizes
4. Add to catalog
5. Submit PR

## Naming Convention

```
category-name-size.extension
logo-company-512x512.png
icon-social-facebook.svg
```

## File Requirements

- Images: Optimized PNG/SVG
- Icons: SVG preferred
- Fonts: Include license
- Media: Compressed formats
